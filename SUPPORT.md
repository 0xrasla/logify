# Support

## Questions / Help

- Use GitHub Issues for bug reports and feature requests.
- For questions, please include:
  - Package name (`@rasla/logify` or `@rasla/express-logify`)
  - Version
  - Runtime (Node/Bun) and OS
  - Minimal reproduction (code snippet)
